Tankyu2 完全オフラインPWA

アップロードするファイル:
- index.html
- manifest.json
- sw.js
- icon-192.png
- icon-512.png

使い方:
1. GitHub Pages の Tankyu2 フォルダに、この5ファイルを上書きアップロード。
2. iPad の Safari で一度オンライン状態で開く。
3. すべてのページを一通り表示する。
4. 共有 → ホーム画面に追加。
5. 以後はホーム画面の Tankyu2 から起動。
